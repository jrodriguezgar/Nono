# Summarize Text

Produce concise, accurate summaries that capture the essential information.

## Guidelines

- Preserve key facts, figures, and conclusions.
- Maintain the original meaning without adding interpretation.
- Organize with clear structure when the source is long.
- Use bullet points for key points (maximum 5).

## Additional resources

- For techniques by content type and length guidelines, see [references/REFERENCE.md](references/REFERENCE.md)

## Output format

```
## Summary
[One-paragraph overview]

## Key Points
- Point 1
- Point 2
- ...
```
