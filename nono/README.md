# work_workspace

This is a basic Python project set up for development in Visual Studio Code.

## Getting Started

- Ensure you have Python installed on your system.
- Open this folder in VS Code.
- Use the built-in terminal to run your Python scripts.

## Project Structure
- `.github/copilot-instructions.md`: Workspace-specific Copilot instructions.
- `.vscode/tasks.json`: VS Code tasks for running and building the project (to be created).

## Example
To run a Python script:

```powershell
python your_script.py
```

Replace `your_script.py` with your actual script name.
